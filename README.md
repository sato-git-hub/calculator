# 電卓を作りました
  
  
## 1.数字を二つ入力  
  
  
### 入力された数字が負の数、小数であった場合
  
`if ! [[ "$num1" =~ ^-?[0-9]+(\.[0-9]+)?$ ]]; then `をコードに加え、負の数と小数の計算ができるようにしました。  
　　
### 入力されたのが、数字（負の数、小数を含む）でない場合  
  
`"Error: '$num1' is not a valid number"`を表示します。  
`exit 1`をコードに加え、プログラムが終了するようにしました。 
  
  
## 2.演算子の選択　　+,-,*(×),/(÷)の中から一つ選び、入力  
  
### 数字の入力画面で 2つ目の数字に0が入力され、演算子の選択画面で /(÷)が入力された場合    
   
`if [[ "$num2" == "0" || "$num2" == "0.0" ]]; then`をコードに加えました。  
条件に当てはまった場合、`echo "Divide by zero error"`を表示します。  
`exit 1`をコードに加え、プログラムが終了します。  
  
### 数字の入力画面で小数が入力された場合  
  
`result=$(echo "$num1 + $num2" | bc)`コードにbcコマンドを加え、小数の計算ができるようにしました。
  
  
 ## 3.結果を出力します
  
結果が小数点の場合、小数2の桁以降の結果は切り捨てられます。
